Loyalty is a cross-domain primitive in CityOS: points accrued in grocery, fitness, pharmacy, or any participating merchant can be redeemed across the platform. The vertical is served by the `loyalty` domain package and exposed at `/api/bff/commerce/loyalty`.

## Endpoints

| Method | Path | Auth | Purpose |
| --- | --- | --- | --- |
| `GET` | `/api/bff/commerce/loyalty` | Required | Balance, tier, programId |
| `POST` | `/api/bff/commerce/loyalty` | Required | Action-based: `redeem`, `history`, `enroll` |

## Actions

| Action | Required fields | Notes |
| --- | --- | --- |
| `enroll` | `programId` | Join a loyalty program |
| `redeem` | `points` (≥ 1) | Spend points |
| `history` | — | Return transaction list |

### Enroll

```bash
curl -X POST https://cityos.dakkah.city/api/bff/commerce/loyalty \
  -H "Authorization: Bearer <token>" \
  -H "x-tenant-slug: riyadh-downtown" \
  -H "Content-Type: application/json" \
  -d '{ "action": "enroll", "programId": "pgm_gold_rewards" }'
```

### Redeem

```bash
curl -X POST https://cityos.dakkah.city/api/bff/commerce/loyalty \
  -H "Authorization: Bearer <token>" \
  -H "x-tenant-slug: riyadh-downtown" \
  -H "Content-Type: application/json" \
  -d '{ "action": "redeem", "points": 500 }'
```

## Account shape

```json
{
  "customerId": "cust_01H...",
  "pointsBalance": 1280,
  "tier": "gold",
  "programId": "pgm_gold_rewards"
}
```
