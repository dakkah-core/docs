### CityOSProviderProps

**Kind:** interface | **File:** `CityOSProvider.tsx`
> CityOSProvider — unified provider wrapping BrandingProvider + TenantProvider. Apps that currently mount both providers individually can swap to a single `<CityOSProvider tenant={...} branding={...}>…</CityOSProvider>` call. The original providers remain exported for backwards compatibility; new surfaces should prefer this unified entry. Mount order matters: TenantProvider is the outer frame (tenant-scoped locale + direction are read by BrandingProvider for theme hydration). Related: docs/reports
