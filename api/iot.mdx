## Endpoints

| Method | Path | Description |
| --- | --- | --- |
| `GET` | `/api/bff/iot/devices` | List IoT devices |
| `GET` | `/api/bff/iot/devices/{id}` | Get device with latest metrics |
| `POST` | `/api/bff/iot/devices/{id}/commands` | Send a command to a device |
| `GET` | `/api/bff/iot/readings` | Query historical readings |
