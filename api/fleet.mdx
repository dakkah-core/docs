## Endpoints

| Method | Path | Description |
| --- | --- | --- |
| `GET` | `/api/bff/fleet/vehicles` | List vehicles |
| `GET` | `/api/bff/fleet/vehicles/{id}` | Get a single vehicle |
| `POST` | `/api/bff/fleet/shipments` | Create a shipment |
| `GET` | `/api/bff/fleet/shipments` | List shipments |
| `POST` | `/api/bff/fleet/shipments/{id}/dispatch` | Dispatch a shipment |
